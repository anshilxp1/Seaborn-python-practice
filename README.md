# Seaborn Tutorial

This repository contains my **Seaborn practice notebook** completed during my **CETPA Infotech Training**.

## 📌 About
The notebook demonstrates the basics of **Seaborn**, a Python data visualization library built on top of Matplotlib. It includes examples of creating different statistical and informative plots.

## 📚 Topics Covered
- Introduction to Seaborn
- Importing Seaborn and Matplotlib
- Loading datasets
- Scatter Plot
- Line Plot
- Bar Plot
- Histogram
- Box Plot
- Violin Plot
- Count Plot
- Pair Plot
- Heatmap
- Customizing plots
- Data visualization using Seaborn

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Seaborn
- Matplotlib
- Pandas
- NumPy

## ▶️ How to Run
1. Clone this repository.
2. Install the required libraries:
   ```bash
   pip install seaborn matplotlib pandas numpy jupyter
   ```
3. Open the notebook:
   ```bash
   jupyter notebook Seaborn_1.ipynb
   ```

## 🎯 Learning Outcome
This notebook helped me understand how to create effective statistical visualizations, customize plots, and analyze data using Seaborn.
